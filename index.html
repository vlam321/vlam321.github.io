Work Experience

Syapse, Inc. - Software Engineer Intern

During the summer of 2017, I was a Software Engineer Intern at Syapse working in the back-end/server team. The company provides a precision medicine platform that allows health-care providers to use molecular profiling to diagnose and treat cancer patients. My primary tasks at Syapse include refactoring Syapse's ontology functionalities to utilize PostgreSQL data-store to enhance horizontal scalability, and developing a load testing framework to run simulated user loads against the system's API endpoints. At the end of the internship, I provided live software demos to my engineering team, as well as to a company-wide audience. Although the duration of this experience was only three months long, I was exposed to and picked up many new computer science concepts and coding practices used in the industry.
